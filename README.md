A keyboard in browser to generate mock data for gliding keyboard impl

1. Install bun:

```bash
curl -fsSL https://bun.sh/install | bash
```

2. Install dependencies:

```bash
bun install
```

3. Start the dev server:

```bash
bun dev
```
